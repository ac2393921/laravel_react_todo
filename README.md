# laravel_react_todo
 
This is Todo App in Laravel with React.
 
# Usage
 
```bash
git clone https://github.com/ac2393921/laravel_react_todo.git
cd laravel_react_todo
composer install
npm install
php artisan key:generate
php artisan jwt:secret
php artisan config:cache
```
